# Feature TDD Demo

### Task:
Create a simple CRUD feature based on a central state management, like recoil.
Implement unit tests for the feature, before the feature is implemented.

### Feature: Create-New-Field Element
- Textfield for the field name
- Button to create the field
- RegExp validation for the field name [a-zA-Z0-9]
- API mock for the field creation

### GOAL:
- TDD showcase
- Separation of Concerns
